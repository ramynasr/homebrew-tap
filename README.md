# Homebrew Tap for linear-for-ai

This is the official Homebrew tap for [linear-for-ai](https://github.com/ramynasr/linear-for-ai), a CLI tool for AI agents to interact with Linear's GraphQL API.

## Installation

```bash
brew tap ramynasr/tap
brew install linear-for-ai
```

## Upgrading

```bash
brew upgrade linear-for-ai
```

## Uninstallation

```bash
brew uninstall linear-for-ai
brew untap ramynasr/tap
```

## Supported Platforms

- macOS (Intel and Apple Silicon)
- Linux (x86_64 and ARM64)

## Documentation

For usage documentation, see the [main repository](https://github.com/ramynasr/linear-for-ai).
