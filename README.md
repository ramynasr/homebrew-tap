# Homebrew Tap

Official Homebrew tap for Ramy Nasr's CLI tools and applications.

## Available Tools

### linear-for-ai

CLI tool for AI agents to interact with Linear's GraphQL API.

- **Repository**: [ramynasr/linear-for-ai](https://github.com/ramynasr/linear-for-ai)
- **Platforms**: macOS (Intel & Apple Silicon), Linux (x86_64 & ARM64)

```bash
brew install linear-for-ai
```

## Installation

First, tap this repository:

```bash
brew tap ramynasr/tap
```

Then install any available tool:

```bash
brew install <tool-name>
```

## Upgrading

Update all tools from this tap:

```bash
brew upgrade
```

Or upgrade a specific tool:

```bash
brew upgrade <tool-name>
```

## Uninstallation

Uninstall a specific tool:

```bash
brew uninstall <tool-name>
```

Remove the tap entirely:

```bash
brew untap ramynasr/tap
```

## Supported Platforms

All tools in this tap support:
- macOS Intel (x86_64)
- macOS Apple Silicon (ARM64)
- Linux x86_64
- Linux ARM64
